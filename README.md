# TrueCare Clinic Website
A simple, responsive health care website built with HTML5, CSS3 and JavaScript, made for the MANTRA 2026 Summer School frontend assignment.

## Pages
- index.html - Home page
- about.html - About the clinic
- services.html - Departments and services, with an FAQ toggle
- gallery.html - Filterable photo gallery
- appointment.html - Appointment booking form, emergency contact and health tip

## Features
- Responsive layout using CSS media queries (mobile, tablet, desktop)
- Basic SEO: title tags, meta description, meta keywords, one h1 per page, alt text on images, canonical tags, sitemap.xml
- Structured data: MedicalClinic schema (JSON-LD) on the home page for better search/AI visibility
- JavaScript features: appointment form validation, gallery filter (including a General Medicine filter), FAQ toggle
- Simple color theme, consistent navigation, and consistent footer (with address) across all pages
- Favicon included
